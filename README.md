## ¿Qué es y que hace la práctica? 
 La práctica realizada consiste en el desarrollo de un detector de spam, capaz de analizar correos electrónicos o mensajes para clasificarlos como spam o normales. Esta herramienta permite identificar con mayor precisión mensajes no deseados, lo que ayuda a prevenir riesgos, evitar fraudes y mejorar la seguridad al no caer en este tipo de correos engañosos.

## ¿Qué aprendimos?
 Durante la realización de esta práctica aprendimos diversos aspectos importantes del desarrollo de software. Entre ellos, el uso de comandos en la terminal CMD, así como la creación y activación de un entorno virtual mediante comandos. También adquirimos conocimientos sobre el manejo de GitHub, incluyendo la creación de repositorios y la subida de archivos a través de la terminal de Visual Studio Code. Además, comprendimos la estructura adecuada que debe llevar un proyecto de software para mantenerlo organizado y funcional.

## ¿Cómo lo hicimos? 
 Este proyecto se llevó a cabo con el apoyo del profesor y mediante el uso de presentaciones realizadas en clase. Gracias a ello, fue posible crear el entorno virtual desde la terminal CMD, y dentro de este entorno, llamado venv, se organizó la estructura del proyecto junto con las librerías necesarias ya instaladas. Posteriormente, se creó el archivo principal donde se desarrolló el código del programa.
 
## Tecnologías-Librerías utilizadas 
 Para el desarrollo del programa utilizamos librerías como Pandas, NumPy y Scikit-Learn, las cuales fueron fundamentales para su correcto funcionamiento. Gracias a estas herramientas, fue posible procesar los datos, realizar cálculos necesarios y aplicar técnicas de aprendizaje automático, logrando así que el detector de spam fuera funcional y eficiente.
